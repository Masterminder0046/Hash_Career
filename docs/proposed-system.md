# Proposed System

The AI Placement Intelligence Platform (PlacementIQ) offers:

- **AI Resume Analysis**: Upload PDF/DOCX, extract skills, ATS scoring, suggestions
- **Company Database**: 100+ companies with requirements and selection process
- **Smart Matching**: Profile vs company requirement comparison with eligibility %
- **ML Prediction**: Random Forest model predicting placement probability
- **Skill Gap Analysis**: Identifies missing skills with learning priority
- **AI Roadmaps**: Personalized 30/60/90 day study plans
- **Mock Interviews**: Text and voice-based interviews with AI evaluation
- **Face Analysis**: Real-time attention metrics using browser APIs
- **Career Chatbot**: AI-powered career guidance
- **Analytics Dashboard**: For students, officers, and administrators
- **Reports**: CSV export, PDF generation, charts
- **Modern UI**: Glassmorphism design with dark/light mode
