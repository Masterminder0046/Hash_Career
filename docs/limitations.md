# Limitations

1. **Demo Mode** - Gemini AI fallback used when API key not configured
2. **Local Storage** - File storage is local (not cloud) in development
3. **No Real-time Video** - Uses browser speech API (no WebRTC video calls)
4. **Face Analysis** - Simplified metrics (not full MediaPipe integration)
5. **Sample Data** - Demo data is synthetically generated
6. **No Payment Gateway** - Not applicable for institutional use
7. **Scalability** - Single MongoDB instance; needs sharding for large scale
8. **Offline Support** - Limited offline capabilities
