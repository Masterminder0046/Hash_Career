# Existing System

Current placement preparation methods include:

1. **Manual Resume Review** - Students get resumes reviewed by seniors or faculty, no AI analysis
2. **Company Websites** - Students manually check company career pages for requirements
3. **Generic Preparation** - One-size-fits-all approach without personalized recommendations
4. **Offline Interviews** - Limited mock interview practice without AI evaluation
5. **Spreadsheet Tracking** - Placement officers use Excel to track student progress
6. **No Central Platform** - Fragmented tools for different aspects of placement prep

These approaches lack personalization, scalability, and data-driven insights.
